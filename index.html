FAQ

How is this different from a formal audit?

Pre-audit security reviews focus on finding and fixing vulnerabilities before formal audits. VectorGuard Labs is more aggressive in its testing, uses advanced attack simulation, and work collaboratively with your team. Formal audits are compliance-focused and occur after your code is production-ready and before mainnet deployment.

What exactly do I get for my 10000 + 1.50/LOC investment?

You'll receive a comprehensive, adversarial, offensive, red-team preliminary security assessment including: red-team testing using up to 338 attack vectors and detailed reports with severity ratings and fix recommendations. 

Do you work with all blockchain platforms or just Ethereum-based blockchains (EVM)?

VectorGuard Labs currently specialize in EVM-compatible chains like Ethereum, BASE, Polygon, Arbitrum, Optimism, BSC, etc. 

What information does VectorGuard Labs need to get started?

VectorGuard Labs need your smart contract source code, documentation about intended functionality, any known dependencies or integrations, and testnet deployment details if available. We can work with private GitHub repos or secure file transfers as well.

How do I know my code and intellectual property will be protected?

VectorGuard Labs can sign a comprehensive NDA before accessing any code and maintain strict confidentiality protocols. Your code is stored in encrypted, access-controlled environments and is deleted after engagement completion.

How long does a typical pre-audit security review take?

Most pre-audit reviews are completed within 72 hours with a full detailed report highlighting the attack vector(s) that successfully broke your code and recommended fixes.

Will my contract go through all 338 attack vectors?

No. Your contract is analyzed and a list of all the attack vectors that affect your protocol directly or indirectly will be used and considered for the preliminary security assessment.

Do you provide a guarantee or warranty on your findings?

While VectorGuard Labs cannot guarantee zero vulnerabilities (no security firm can), VectorGuard Labs guarantee comprehensive red-team, offensive, attack testing using industry-leading methodologies.

If my test comes back positive, can I deploy my contract to mainnet?

No, VectorGuard Labs strongly recommend proceeding to a formal third-party audit before mainnet deployment. VectorGuard Labs is designed to be the critical step between contract development and formal audit.  

Think of VectorGuard's pre-audit red-team testing as quality assurance that ensures you won't waste $50,000+ on a formal audit that comes back with dozens of findings. Once your contract is secure, it means you're ready for the formal audit process with confidence. The formal audit provides the final compliance certification, documentation, and industry credibility needed for mainnet launch, especially if you're seeking investment, partnerships, or handling significant user funds.

VectorGuard Labs' clean red-team pre-audit results dramatically improve your formal audit experience—fewer findings, faster completion, lower costs, and stronger auditor confidence in your security practices.
